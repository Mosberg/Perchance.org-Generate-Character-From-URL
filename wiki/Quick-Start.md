# Quick Start

Two ways to get started: generate one character or process many at once.

---

## Generate a single character

1. Open [Generate Character From URL v6](https://perchance.org/generate-character-from-url)
2. Paste a URL into **Source URL**
3. _(Optional)_ Add instructions in **Optional extra instructions**, e.g. `Make the tone more formal`
4. Click **Generate character**
5. Wait for the progress bar to finish
6. Review and edit fields in the **Generated Character** panel
7. Click **Download JSON**
8. In ai-character-chat, use **Import** → select the downloaded file

That's it. The character is ready to chat.

---

## Generate many characters at once

1. Paste URLs into the **Batch URL** textarea — one per line
2. In **Queue export mode + global settings**, set **JSON output mode** to `One combined file only`
3. Click **Start queue**
4. When the run finishes, download the combined JSON
5. Import once into ai-character-chat — all characters appear together

---

## Tips for better results

- Pages with rich personality descriptions and dialogue give better output
- Add extra instructions to steer tone, length, or focus
- Use the **Extraction details** panel at the bottom to debug weak results
- If a source fails, check [[Troubleshooting]]
